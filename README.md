# BME-Labor-Portfolio

Ebben a repozitóriumban az egyetemi tanulmányaim során elvégzett legfontosabb projektjeimet és laborfeladataimat gyűjtöttem össze. A portfólió három fő pillérre épül: infrastruktúra, szoftverfejlesztés és minőségbiztosítás.

## Szakmai projektek és laborok

### 1. Nyílt forráskódú projektfejlesztés (CI/CD páros munka)
Egy nagyméretű, open source szoftver funkcionális bővítésén dolgoztam páros munkában. A fejlesztés során a teljes **CI/CD életciklust** alkalmaztuk:
* **Specifikáció és Implementáció:** Magas minőségű, dokumentált kód készítése.
* **Black-box tesztelés:** Funkcionális validáció az implementáció ismerete nélkül.
* **Code Review:** Szigorú szakmai ellenőrzés a Clean Code elvek mentén.

### 2. Mobil- és Webfejlesztés
* **Android (Kotlin):** Hét projektet valósítottam meg **Jetpack Compose** és modern **MVVM** architektúra használatával (Retrofit, Room adatbázis).
* **Webes technológiák:** Hat webes labor keretében **HTML5, CSS3, JavaScript és jQuery** használatával készítettem kliensoldali megoldásokat és reszponzív felületeket.
